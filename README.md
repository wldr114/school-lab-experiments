# 链路状态路由协议仿真

计算机网络课程设计 — 基于 Python 的链路状态路由协议 (Link State Routing Protocol) 仿真程序。

## 功能

- 自定义网络拓扑（节点 + 链路 + 代价），JSON 文件存储
- 多线程模拟分布式路由器，`queue.Queue` 实现线程间 LSP 消息交换
- Dijkstra 最短路径算法计算路由表
- 动态修改链路代价 / 增删链路 / 增删节点，自动触发路由更新
- tkinter 可视化界面：拓扑图 + 路由表 + 控制面板

## 运行

```bash
python main.py
```

启动后点击**加载拓扑**，选择 `topology.json`。

## 项目结构

```
├── main.py          # 程序入口
├── lsp.py           # 链路状态分组 (LSP) 数据结构
├── topology.py      # 拓扑管理 — 邻接表 + JSON 读写
├── dijkstra.py      # Dijkstra 最短路径算法 + 路由表生成
├── router.py        # 路由器线程 + 网络模拟器
├── gui.py           # tkinter 可视化界面
└── topology.json    # 拓扑配置文件
```

## 拓扑配置格式

```json
{
  "nodes": ["A", "B", "C", "D", "E"],
  "edges": [
    {"from": "A", "to": "B", "cost": 4},
    {"from": "A", "to": "C", "cost": 2}
  ]
}
```

## 操作说明

| 操作 | 方式 |
|---|---|
| 加载拓扑 | 按钮 → 选择 JSON 文件 |
| 修改链路代价 | 右键点击边 → 修改链路代价 |
| 删除链路 | 右键点击边 → 删除链路 |
| 添加链路 | 按钮 → 选择两端节点和代价 |
| 添加/删除节点 | 对应按钮 |
| 保存拓扑 | 保存拓扑按钮 |
| 刷新路由表 | 刷新路由表按钮 |

## 环境

- Python 3.x（标准库，无需额外安装依赖）
